Reddit Bot: Rock, Paper, Scissors

Resolve disputes with civility!
Utilized: Python Reddit API Wrapper (PRAW), Python, Heroku

Process:

  • Bot replies to user mentions to initiate a 'Rock, Paper, Scissors' match.
  • Parties to the match then private message the bot as prompted.
  • After 5 minutes (or the above) the bot replies with the result.
  • Now deployed to Heroku!
